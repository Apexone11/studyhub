Feed Card + Comments Redesign

Facebook-inspired redesign building on existing StudyHub structure. Click to select your preferred approach.

A

Approach A: Facebook Classic

Closest to Facebook's layout. Full-width card, author row on top, content below, stats bar (likes/comments/shares counts), then action bar (Like | Comment | Share), comments below with nested replies.

  • Stats bar shows "X likes - Y comments - Z shares" as subtle text links
  • Action bar uses icon+label buttons separated by dividers
  • Comments open inline below the action bar
  • Replies nest up to 3 levels with left-indent + connecting line
  • Like/Dislike icons are solid SVGs, fully visible
B

Approach B: Facebook + StudyHub Identity

Facebook layout structure but keeps StudyHub's card styling, brand colors, and type badges (POST, SHEET, NOTE). Familiar but distinct.

  • Same stats/action bar pattern as Facebook
  • Keeps StudyHub's rounded cards with subtle border
  • Type pill badges remain (POST, SHEET, NOTE, ANNOUNCEMENT)
  • Brand color accents on active Like/Dislike states
  • Comments use the same Facebook threading but with StudyHub's color tokens
C

Approach C: Minimal Facebook Influence

Only adopt Facebook's comment threading and spacing. Keep the current feed card layout mostly intact, just fix icons and polish spacing.

  • Current card layout stays (avatar left, content right)
  • Only comment section gets the Facebook treatment
  • Fix like/dislike icons to show fully
  • Better spacing between elements
  • Less visual change overall

Key Design Decisions

  • Stats bar between content and action buttons (like Facebook)
  • Action bar with full icon + label: Like | Dislike | Comment | Share
  • Comment bubbles with rounded pill shape (Facebook-style)
  • Reply actions as text links: Like - Dislike - Reply - timestamp
  • 3-level nesting with left border + decreasing avatar sizes (34px > 28px > 24px)
  • No reply button at level 3 - replies flatten into the deepest thread
  • Share button copies a link (no external sharing yet)
  • StudyHub identity preserved: type badges, brand colors, card styling
