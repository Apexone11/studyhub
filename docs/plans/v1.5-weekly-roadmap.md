## Design Direction: Clean Academic Pro

### Core Principles
1. **Subtle + Purposeful Motion**: Anime.js for meaningful transitions, not decoration
2. **Light-first + Dark-compatible**: Default light theme with proper dark mode support
3. **Accessibility**: WCAG 2.2 AA strict - keyboard nav, screen readers, color contrast
4. **Responsive-first**: Mobile breakpoints tested on every page
5. **Consistent Spacing**: 8px grid system (8, 16, 24, 32, 48, 64)
6. **Typography Scale**: Plus Jakarta Sans, consistent size hierarchy

### Color Palette
- Primary: #2563eb (Blue 600)
- Primary Dark: #1e40af (Blue 800)
- Background: #f8fafc (Slate 50)
- Surface: #ffffff
- Border: #e2e8f0 (Slate 200)
- Text Primary: #0f172a (Slate 900)
- Text Secondary: #64748b (Slate 500)
- Success: #16a34a (Green 600)
- Warning: #f59e0b (Amber 500)
- Error: #dc2626 (Red 600)
- Gradient: linear-gradient(135deg, #1e3a5f, #2563eb)

### Animation Patterns
- Page enter: fadeInUp (300ms, easeOutQuad)
- Card stagger: staggerEntrance (50ms delay between items)
- Button hover: popScale (1.03x, 200ms)
- Stat count: countUp (800ms, easeOutExpo)
- Scroll reveal: fadeInOnScroll (IntersectionObserver threshold 0.15)
