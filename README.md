# StudyHub

StudyHub is a collaborative study platform that gives students a GitHub-style way to create, share, fork, improve, and organize study materials by course.

Built by students, for students, StudyHub turns scattered notes, dead group chats, and hard-to-find review files into a shared knowledge base that can keep improving over time.

## Key Capabilities

- GitHub-style study sheet sharing with forks, stars, comments, and contribution flows
- Public feed posts with expandable discussions, reactions, mentions, and notifications
- Course-based discovery so students can find the right material faster
- Private markdown notes linked to your classes
- Optional file attachments with creator-controlled downloads
- Verified email, password reset, and 2-step verification for account security
- Student-first experience with no paywalls and no third-party ads

## How It Works

1. Create your account and choose your school and courses.
2. Browse study sheets from classmates and fork the ones you want to improve.
3. Post updates, comment on work, mention other users, and build stronger materials together.
4. Contribute your improvements back so the original creator can review and accept them.

## Roadmap

### Version 2

## Why StudyHub

- Open access: study materials stay free to use
- Student collaboration: notes get better when more people can improve them
- Start local, grow outward: begin at Maryland, then expand campus by campus
- Privacy-first mindset: collect only what is needed to run the platform well
